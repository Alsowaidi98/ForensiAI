How it works (high-level)

ForensiAI is written in Python using Streamlit, Pandas and a small SQLite state database. It is designed to be clear enough for examiners to understand but realistic enough to look like a small SOC tool.

1. Ingest

Log collectors export events to CSV (macOS unified log, Windows security & Sysmon). ForensiAI ingests these into a unified data model.

2. Detect

Simple but explainable detectors flag port-scans, failed logins, suspicious processes and persistence changes. Events are scored and ranked.

3. Explain & report

Analysts review the ranked events and generate PDF reports summarising hosts, timelines and indicators ‚Äì perfect for attaching to the thesis.
