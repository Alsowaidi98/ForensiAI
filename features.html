What ForensiAI can do for incident demos.

ForensiAI combines cybersecurity logging, AI-style risk scoring, data analytics and a Project Manager console. It is designed to look like a small SOC tool while staying realistic for a capstone project.

Built with Python, Streamlit, macOS Unified Log & Windows event logs, and SQLite ‚Äì with clear separation between the analyst view and the management view.
